Use the following commands when running the Vending Machine program from the Windows command prompt.

--------------------------
Open up the src directory:
--------------------------
cd src

----------------------------------
Compile each Java file separately:
----------------------------------
javac VendingMachine\Drink.java
javac VendingMachine\Machine.java
javac Main.java

---------------------------
Execute the Main.java file:
---------------------------
java Main

--------------------------------------------------
Purchase a drink from the virtual vending machine:
--------------------------------------------------
Follow the on-screen instructions by entering in a specific amount of money and a drink selection number.
